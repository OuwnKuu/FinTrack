# FinTrack: Aplikasi Pencatat Keuangan Yang Multi-Account

Aplikasi ini sebenarnya adalah tugas. Tapi saya rasa ini bisa jadi bahan belajar untuk menggunakan Git dan GitHub.
Aplikasi ini dibuat menggunakan Android Studio dengan XML untuk UI dan Kotlin sebagai bahasa utama.

**Fitur-Fitur:**
1. Multi-Account
2. Fitur CRUD di data keuangan (buat catatan baru, lihat catatan beru secara otomatis, edit catatan yang sudah ada, dan hapus catatan)
3. Fitur CRUD di manajemen akun
   
**Cara memulai (Getting Started):**

Dimulai dari mendapatkan folder project:
1. Bisa lewat Git Clone `https://github.com/OuwnKuu/FinTrack.git`
2. Bisa download file .zip

**Dengan Android Studio**

Buka aplikasi Android Studio dengan versi Gradle minimal 8.13 untuk melakukan build & run. Colokkan kabel USB dari komputer ke HP (USB debugging). Lakukan run dengan device HP yang terpilih.
